# Uber-Ola-fare-prediction-using-ml
I have done a project on Ola and uber fare estimation.The code consists of 5 phases preprocessing,spliting,training,testing and evaluation.
I have trained ml models like linear regression for uber fare prediction and then tested the models by predicting fares of test dataset.I have used xgboost classifier to predict fare of ola cabs.
Accuracy for both ml models has been mentioned  in code.
